# 数据库

- #### 设置字符集
use names utf8


# Git
### stash功能
- #### 暂存
git stash 保存在git整个栈中
- #### 设置保存名
git stash save "stashName"
- #### 查看暂存列表
git stash list
- #### 恢复最近一次暂存，从暂存列表中删除
git stash pop
- #### 应用某一次暂存，暂存列表中不删除
git stash aplly "stashName"
