# 数据库
### MySql
- #### 连接数据库
 mysql -h host -u username -p password
- #### 设置字符集
use names utf8

### Redis
- #### 连接数据库
redis-cli -h host -p port -a "password"
- #### 查询所有可用的key
key *
- #### 选择数据库
select db

# Git
### stash功能
- #### 暂存
git stash 保存在git整个栈中
- #### 设置保存名
git stash save "stashName"
- #### 查看暂存列表
git stash list
- #### 恢复最近一次暂存，从暂存列表中删除
git stash pop
- #### 应用某一次暂存，暂存列表中不删除
git stash aplly "stashName"


# NodeJs
### hexo
- #### 新建一个页面
hexo new "pageName"
- #### 新建一个草稿
hexo new draft "draftName"
- #### 生成静态文件
hexo generate/hexo g
- #### 启动hexo服务
hexo server/hexo s
- ##### 部署发布
hexo deploy/hexo d
